## 如何使用

1. 克隆本仓库：
   ```bash
   git clone https://github.com/gpboyer1/weird-worm-p5.git
   ```
2. 双击打开 `main.html` 文件，即可在浏览器中看到动画效果。

## 主要文件

- `main.html`：包含全部动画代码和 p5.js 引用。
